# LunaShade — Release Notes

## v1.0.0 — June 20, 2026

Initial public release.

### Highlights

- Premium glass-inspired popup UI with smooth animations.
- Smart dark theme engine that preserves images, videos, SVG, and canvases.
- Per-site control with global or allowlist-only modes.
- Adjustable brightness (50–150%) and contrast (50–150%) with live preview.
- Exclusion list with one-click add/remove.
- Settings sync across your Chrome devices.
- Zero telemetry. Zero external dependencies.
- Manifest V3, minimal permissions.
