Privacy Policy

LunaShade — Elegant Dark Mode

Effective Date: June 2026

Overview

This Privacy Policy explains how the LunaShade Chrome Extension operates with respect to user data. By using the extension, you agree to the practices described below.

Data Collection

LunaShade does not collect, store, transmit, or sell any personal data. The extension operates entirely within the user’s browser.

Website Content Processing

The extension temporarily accesses and modifies website content (such as colors, styles, and layout) in order to apply dark mode functionality. This processing occurs locally on the user’s device and is not transmitted to any external servers.

Permissions Usage

  • activeTab: Used to apply dark mode when the user activates the extension.
  • storage: Used to save user preferences such as theme settings.
  • scripting: Used to inject styling scripts into web pages for visual modification.

Data Sharing

LunaShade does not share any data with third parties, advertisers, or external services.

Data Security

Since no user data is collected or transmitted, all processing remains fully local within the browser.

Changes to This Policy

This Privacy Policy may be updated in the future to reflect changes in functionality. Any updates will be published at the same URL.
